# Research Articles

## Setup

For first time use run: 

`sh -c "$(curl -fsSL https://raw.githubusercontent.com/mcnichol/research-articles/master/run-me.sh)"`

For each subsequent use:
* Navigate to working directory `research-articles`
* `./run-me.sh`
